RTL-RFM
=======
FSK/GFSK Decoder for RTL-SDR

Usage
-----

```
RTL_RFM, (C) Ryan Suchocki

Usage: rtl_rfm [-hsqd] [-f freq] [-g gain] [-p error] 

Option flags:
  -h    Show this message
  -s    Read input from stdin
  -q    Quiet. Only output good messages
  -d    Show Debug Plot
  -f    Frequency [869412500]
  -g    Gain [50]
  -p    PPM error [47]
```
